# FocusTab – Animated Productivity New Tab

A modern Chrome/Edge extension that replaces your new tab with a beautiful productivity dashboard.

## Features
- ⏰ Live clock
- 🎯 Daily focus tracker
- 🔥 Streak system
- 🎨 Animated UI
- ⚡ Lightweight & fast

## Tech Stack
- HTML
- CSS
- JavaScript
- Chrome Extension API

## Installation
1. Clone repo
2. Load unpacked in Chrome extensions
